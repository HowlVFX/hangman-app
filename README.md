## Run Locally

Clone the project

```bash
  git clone https://github.com/HowlVFX/hangman-app.git
```

Go to the project directory

```bash
  cd hangman-app
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  streamlit run streamlit_app.py
```
